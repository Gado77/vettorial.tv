# VETTORIAL .TV

Site institucional da **VETTORIAL .TV** — Mídia digital estratégica em Picos, Piauí.

## 🚀 Deploy

Este projeto é um site estático (HTML + CSS + JS) pronto para deploy no **Vercel**.

### Vercel
1. Conecte este repositório no [Vercel](https://vercel.com)
2. O Vercel detecta automaticamente como **Static Site** (Other)
3. **Framework Preset:** Other
4. **Output Directory:** `./` (raiz)
5. Pronto! Deploy automático a cada push.
